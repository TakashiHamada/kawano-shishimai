# 稽古の音八 - 川野獅子舞 笛練習専用プレイヤー

川野獅子舞の笛の練習のための音声プレイヤーです。

## 機能

- 音声の再生・一時停止
- 5秒巻き戻し
- ループ範囲設定（開始点・終了点を指定して部分ループ再生）
- 再生速度変更（0.6x / 0.8x / 1.0x）
- 選択した音源をクッキーに保存

## 使い方

ウェブブラウザからアクセスし、セレクトボックスから音声を選んで再生してください。

## 音声の追加方法

1. MP3ファイルを `audio/` フォルダに配置
2. `audio_metadata.csv` にエントリを追加

### CSVフォーマット

```
filename,title
```

- **filename**: 音声ファイル名（半角英数字・ハイフン・アンダースコアのみ）
- **title**: 表示タイトル

※ 音声ファイルはMP3形式のみ対応です。
